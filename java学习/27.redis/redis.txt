Redis:高性能的NOSQL系列的非关系型数据库
	1.概念：什么是nosql
		nosql：not only sql泛指非关系型数据库
		关系型数据库和非关系型
			1.关系：
				数据之间有关联关系（员工表，部门表）
				数据存储在硬盘文件上
			2.非关系
				没有
				内存
	2.命令操作
		redis存储的是key-value格式的数据，key为字符串，value有五种不同的数据结构
			1.string
				存储：set key value
				获取：get key
				删除：del key
			2.map
				存储：hset key field value
				获取
					hget key field
					hgetall key
				删除：hdel key field
			3.linkedlist(双端队列)
				存储
					lpush key value
					rpush
				获取：lrange key 0 -1
				删除
					lpop key
					rpop
			4.set
				存储：sadd key value
				获取：smembers key
				删除：srem key value
			5.sortedset(以分数来排序)
				存储：zadd key score value
				获取：zrange key 0 -1
				删除：zrem key value
				
				通用命令：
					1.key *
					2.type key
					3.del key
	
	3.持久化
		1.RDB（redis database）：默认方式，在一定间隔时间内检测key的变化，然后持久化
			配置redis.windows.config
				//900秒钟至少一个key发生变化
				1.save 900 1
			重启服务器，指定配置文件名称
		2.AOF(append only file):日志记录方式，可以记录每一条命令操作
		
jedis: java操作redis的工具
	1.使用
		获取连接
			new Jedis(ip,端口)
		操作
		关闭连接
	2.操作
		setex(key,second,value): 多少秒后自动删除（设置激活码）
